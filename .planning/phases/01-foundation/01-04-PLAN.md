Create console module for Rich integration:

**Console instance:**
- Create module-level `console = Console()`
- Export for use across CLI

**Helper functions:**

`print_table(title: str, columns: list[str], rows: list[list[str]]) -> None:`
- Create Rich Table with title
- Add columns from list
- Add rows
- Print via console

`print_error(message: str) -> None:`
- Print with red styling: `[red]Error:[/red] {message}`

`print_success(message: str) -> None:`
- Print with green styling: `[green]✓[/green] {message}`

`print_warning(message: str) -> None:`
- Print with yellow styling: `[yellow]Warning:[/yellow] {message}`

`get_progress() -> Progress:`
- Return configured Progress instance for file operations
- Columns: SpinnerColumn, TextColumn, BarColumn, TaskProgressColumn

Do NOT create abstract "themes" or config systems.
Do NOT add emoji toggles or complex formatting options.
Keep it simple - these are utility functions.
