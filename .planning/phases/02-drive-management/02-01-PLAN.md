  <name>Task 1: Create drives module with drive detection</name>
  <files>src/drivecatalog/drives.py</files>
  <action>
Create drives.py module with functions for detecting drive information on macOS:

1. `get_drive_uuid(path: Path) -> str | None` - Run `diskutil info -plist <path>` and parse VolumeUUID from plist output. Return None if not found or command fails.

2. `get_drive_size(path: Path) -> int` - Use `os.statvfs(path)` to get total bytes (f_frsize * f_blocks).

3. `get_drive_info(path: Path) -> dict` - Combine above into dict with keys: uuid, total_bytes, name (from path.name), mount_path (str of path).

4. `validate_mount_path(path: Path) -> bool` - Check path exists, is directory, and is under /Volumes/ (macOS convention).

Use subprocess.run with capture_output=True for diskutil. Parse plist with plistlib.loads(). Handle errors gracefully - return None for UUID if diskutil fails (non-mounted or non-disk paths).
  </action>
  <verify>python -c "from drivecatalog.drives import get_drive_info, validate_mount_path; print(validate_mount_path('/Volumes'))"</verify>
  <done>Module imports successfully, validate_mount_path returns bool, get_drive_info returns dict with expected keys</done>
</task>
