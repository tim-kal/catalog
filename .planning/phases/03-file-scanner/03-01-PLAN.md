<task type="auto">
  <name>Task 1: Create scanner module with directory traversal</name>
  <files>src/drivecatalog/scanner.py</files>
  <action>
Create scanner.py with:

1. `scan_drive(drive_id: int, mount_path: str, conn: sqlite3.Connection) -> ScanResult` function:
   - Use os.walk() to traverse all directories under mount_path
   - For each file: collect path (relative to mount_path), filename, size_bytes (os.stat), mtime (os.stat as ISO string)
   - Skip hidden files/directories (starting with .)
   - Skip system directories: .Spotlight-V100, .fseventsd, .Trashes, .TemporaryItems
   - Handle permission errors gracefully (log and skip)

2. Database operations within scan_drive:
   - For each file, check if exists in DB (SELECT by drive_id + path)
   - If not exists: INSERT new file record
   - If exists and (size OR mtime changed): UPDATE the record
   - If exists and unchanged: skip
   - Track counts: new_files, modified_files, unchanged_files, errors

3. `ScanResult` dataclass with fields: new_files, modified_files, unchanged_files, errors, total_scanned

Use pathlib.Path for path operations. Store path as relative to mount_path (not absolute) for portability.
  </action>
  <verify>python -c "from drivecatalog.scanner import scan_drive, ScanResult; print('Import OK')"</verify>
  <done>scanner.py exists with scan_drive function that traverses directories and performs DB operations</done>
