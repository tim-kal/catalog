  <files>src/drivecatalog/hasher.py, pyproject.toml</files>
  <action>
1. Add xxhash to project dependencies (pyproject.toml or requirements)

2. Create hasher.py with:
   - `CHUNK_SIZE = 64 * 1024` constant (64KB)
   - `compute_partial_hash(file_path: Path, size_bytes: int) -> str` function:
     * If size_bytes < CHUNK_SIZE * 2 (< 128KB): read entire file
     * Else: read first 64KB + seek to end - 64KB + read last 64KB
     * Create xxhash.xxh64() hasher
     * Update with chunks read
     * Update with str(size_bytes).encode() to include size in hash
     * Return hasher.hexdigest()
   - Handle file read errors gracefully (return None or raise)

3. Use pathlib.Path for file operations. Open in binary mode ('rb').
  </action>
