<task type="auto">
  <name>Task 1: Create duplicates module with clustering queries</name>
  <files>src/drivecatalog/duplicates.py</files>
  <action>
Create duplicates.py with two functions:

1. `get_duplicate_clusters(conn) -> list[dict]`:
   - Query files grouped by partial_hash WHERE partial_hash IS NOT NULL HAVING COUNT(*) > 1
   - Join with drives table to get drive_name for each file
   - Return list of clusters, each containing:
     - partial_hash: str
     - files: list of {drive_name, path, size_bytes}
     - count: int (number of copies)
     - size_bytes: int (size of one copy)
     - reclaimable_bytes: int (size_bytes * (count - 1))
   - Order clusters by reclaimable_bytes DESC (most impactful first)

2. `get_duplicate_stats(conn) -> dict`:
   - Return aggregate statistics:
     - total_clusters: int (number of unique duplicate groups)
     - total_duplicate_files: int (sum of all copies across all clusters)
     - total_bytes: int (sum of size_bytes * count for all clusters)
     - reclaimable_bytes: int (sum of size_bytes * (count - 1) for all clusters)

Use efficient SQL with GROUP BY and aggregate functions. Avoid N+1 queries.
  </action>
