  <name>Task 1: Create search module with query function</name>
  <files>src/drivecatalog/search.py</files>
  <action>
Create search module with `search_files(conn, pattern, *, drive_name=None, min_size=None, max_size=None, extension=None, limit=100)` function.

Implementation:
- Use SQL LIKE with wildcards for pattern matching (convert user pattern to SQL: `*` → `%`, `?` → `_`)
- Match against the `path` column (relative path)
- Optional filters applied via WHERE clauses:
  - drive_name: JOIN with drives table, filter by d.name
  - min_size/max_size: filter by size_bytes
  - extension: filter by path LIKE '%.ext'
- Return list of dicts with: path, size_bytes, drive_name, modified_at
- ORDER BY modified_at DESC (most recent first)
- LIMIT to prevent huge result sets

Do NOT use fnmatch (keep it pure SQL for efficiency).
  </action>

```bash

conn = get_connection()
# Should return empty list on empty DB

print(f'search_files works: {isinstance(results, list)}')
conn.close()
"
```
  </verify>
