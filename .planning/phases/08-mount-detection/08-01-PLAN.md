  <name>Task 1: Create watcher module with VolumeEventHandler</name>
  <files>src/drivecatalog/watcher.py</files>
  <action>
Create watcher.py module implementing /Volumes monitoring:

1. VolumeEventHandler(FileSystemEventHandler):
   - __init__(self, db_path, on_mount_callback, on_unmount_callback)
   - on_created: Check if DirCreatedEvent, filter hidden dirs (name.startswith('.')), call on_mount_callback with path
   - on_deleted: Check if DirDeletedEvent, filter hidden dirs, call on_unmount_callback with path

2. start_volume_watcher(handler) -> Observer:
   - Create Observer instance
   - Schedule handler on "/Volumes" with recursive=False
   - Start observer
   - Return observer for caller to manage

3. get_mounted_volumes() -> list[Path]:
   - List /Volumes directory
   - Filter out hidden entries (startswith('.'))
   - Return list of Path objects for startup race handling

4. run_watcher(db_path, on_mount, on_unmount):
   - Main entry point for CLI
   - Get existing volumes on startup
   - Create handler with callbacks
   - Start observer
   - Set up SIGTERM/SIGINT handlers for graceful shutdown
   - Loop with observer.join(timeout=1) for responsive shutdown

Use pathlib.Path for all paths. Import FileSystemEventHandler, DirCreatedEvent, DirDeletedEvent from watchdog.events, Observer from watchdog.observers.
