     Query parameters:
     - drive (str, optional): Filter by drive name
     - path_prefix (str, optional): Filter by path prefix (directory browsing)

     - min_size (int, optional): Minimum file size in bytes
     - max_size (int, optional): Maximum file size in bytes
     - has_hash (bool, optional): Filter by hash presence
     - is_media (bool, optional): Filter by media flag

     ```sql

     FROM files f
     JOIN drives d ON f.drive_id = d.id

     LIMIT ? OFFSET ?
     ```

     Return FileListResponse with files, total count, page, page_size

  3. GET /files/{file_id} - Get single file details:

     Query parameters (required):
     - q (str): Search pattern (glob-style: *.mp4, *vacation*)

     Query parameters (optional):
     - drive (str): Filter by drive name
     - min_size (int): Minimum file size in bytes
     - max_size (int): Maximum file size in bytes
     - extension (str): Filter by extension (without dot)
     - limit (int, default=100, max=1000): Maximum results

     Use search_files() from search module:
     ```python
     from drivecatalog.search import search_files

     results = search_files(

         min_size=min_size,
         max_size=max_size,
         extension=extension,
         limit=limit,
     )
     ```

     Return SearchResultResponse with files list, total, and pattern
