Create project.yml for xcodegen with:
- Name: DriveCatalog
- Platform: macOS
- Deployment target: 14.0
- Type: application
- Sources: DriveCatalog/
- Info.plist: DriveCatalog/Info.plist
- Settings: SWIFT_VERSION 5.9, CODE_SIGN_STYLE Automatic

Create DriveCatalog/ directory with:
1. DriveCatalogApp.swift - @main App with WindowGroup containing ContentView
2. ContentView.swift - Basic placeholder with "DriveCatalog" text
3. Info.plist - Standard macOS app plist with bundle ID com.drivecatalog.app

Do NOT add Assets.xcassets yet - keep minimal for project foundation.
  </action>
