  <name>Task 1: Create APIService with drives endpoints</name>
  <files>DriveCatalog/Services/APIService.swift</files>
  <action>
Create APIService as an actor for thread-safe networking.

Structure:
- Static baseURL = "http://localhost:8000"
- Custom JSONDecoder with:
  - keyDecodingStrategy = .convertFromSnakeCase
  - dateDecodingStrategy = .custom to parse ISO8601 with optional fractional seconds
- Custom JSONEncoder with:
  - keyEncodingStrategy = .convertToSnakeCase

Implement these methods:
- fetchDrives() async throws -> DriveListResponse
- createDrive(path: String, name: String?) async throws -> DriveResponse
- deleteDrive(name: String) async throws (appends ?confirm=true)
- fetchDriveStatus(name: String) async throws -> DriveStatusResponse

Error handling:
- Create APIError enum: invalidURL, requestFailed(Error), invalidResponse, httpError(Int, String), decodingFailed(Error)
- Parse error responses from API (JSON with "detail" field)

Use URLSession.shared for requests. Add Content-Type: application/json header for POST requests.
